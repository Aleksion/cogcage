# COPY GUIDE — The House Voice

## The Voice

All UI copy is written by The House. The House is not trying to be funny. That's what makes it funny.

The House is:
- Ancient
- Neutral
- Aware of everything
- Slightly ominous
- Completely matter-of-fact about things that should not be matter-of-fact

The House is not:
- Friendly
- Encouraging
- Apologetic
- Using the word "awesome"
- Using exclamation points (one exception: error states, used ironically)

---

## The Register

**Dry. Flat. Committed to the bit.**

Good:
> *"Your Lobster is resting. Don't let it rest too long."*
> *"Other Lobsters are waiting. They're not nervous. You probably are."*
> *"The Pit doesn't care about your feelings. The Pit just keeps filling up."*

Bad:
> "Get ready to battle! 🦞"
> "Welcome back, Chef!"
> "You're doing great!"

## Error States

Errors are the one place The House is slightly more direct. Still not friendly.

| State | Copy |
|-------|------|
| Connection lost | *"The Brine is quiet. Check your connection."* |
| Webhook timeout | *"Your agent didn't respond. The Pit did not wait."* |
| Invalid URL | *"That is not a valid address. The House checked."* |
| No Molts left | *"You need a Molt to enter a Scuttle. Head to The Shed."* |

## Things We Never Say

- "Oops!" 
- "Uh oh!"
- "Awesome"
- "Amazing"
- "Level up"
- "Experience points" / "XP"
- "Craft" (Lobsters don't craft, they build Molts)
- "Equip" (you don't equip a Molt, you *drop your Lobster in*)
- "Play" (you don't play a Scuttle, you *enter* one)
- "User"
- "Game mode"
- "Tutorial" (it's *The Soft Shell Guide*)
